# 3級 新セクション構成

実際の試験比重（85%図表読み取り、15%その他）に基づく構成

## セクション一覧

### Section1: 記述統計とグラフ読解 ★★★★★（最重要・30%）

**サブセクション1** (10問) - ヒストグラムと箱ひげ図
- ヒストグラムから中央値、四分位数、平均値が含まれる階級を読み取る
- 箱ひげ図から五数要約を読み取る
- 複数のヒストグラムの比較（分布の形状、ばらつき）

**サブセクション2** (10問) - 度数分布表と統計量計算
- 度数分布表から平均値、中央値、標準偏差を計算
- 累積相対度数の計算
- 度数分布表と箱ひげ図の対応

**サブセクション3** (10問) - 幹葉図と外れ値
- 幹葉図から中央値、範囲を読み取る
- 外れ値の検出と影響
- データの変換前後での統計量の変化

AIプロンプト例：
```
セクション: 記述統計とグラフ読解
級: 3級
問題数: 10
詳細: 必ずヒストグラムまたは箱ひげ図を含め、中央値や四分位数を読み取る問題を作成してください。
```

---

### Section2: 散布図・相関・単回帰分析 ★★★★★（最重要・25%）

**サブセクション1** (10問) - 散布図と相関係数
- 散布図から相関の強さと方向を判断
- 相関係数の値を推定
- 外れ値が相関係数に与える影響

**サブセクション2** (10問) - 線形単回帰分析
- 回帰式から予測値を計算
- 決定係数の解釈
- 残差の意味と計算

**サブセクション3** (10問) - データ変換と相関
- 1次変換が相関係数に与えない影響
- 単位変換と相関
- 共分散の計算

AIプロンプト例：
```
セクション: 散布図・相関・単回帰分析
級: 3級
問題数: 10
詳細: 必ず散布図を含め、相関係数を推定する問題や、回帰式から予測値を計算する問題を作成してください。重回帰分析は含めないでください。
```

---

### Section3: データの変換と統計量 ★★★★（重要・15%）

**サブセクション1** (10問) - 1次変換の影響
- ax+b変換が平均値、標準偏差に与える影響
- 偏差値の計算
- 標準化（z得点）

**サブセクション2** (10問) - 変動係数と比較
- 変動係数を用いたばらつきの比較
- 平均が異なるデータのばらつき比較
- データの尺度と統計量

AIプロンプト例：
```
セクション: データの変換と統計量
級: 3級
問題数: 10
詳細: データを1次変換（ax+b）したときに統計量がどう変化するか問う問題や、偏差値を計算する問題を作成してください。
```

---

### Section4: 確率と確率分布 ★★★（通常・10%）

**サブセクション1** (10問) - 基本的な確率
- 独立事象の確率
- 条件付き確率
- 余事象の確率
- 組合せ（nCr）の計算

**サブセクション2** (10問) - 二項分布
- 二項分布の基本計算
- 反復試行の確率
- 期待値の計算

AIプロンプト例：
```
セクション: 確率と確率分布
級: 3級
問題数: 10
詳細: 独立事象や条件付き確率、二項分布の基本計算問題を作成してください。ベイズの定理は含めないでください。
```

---

### Section5: 推測統計の基礎 ★★★（通常・10%）

**サブセクション1** (10問) - 標本分布と信頼区間
- 中心極限定理の概念
- 標本平均の期待値と分散
- 信頼区間の概念と計算
- 信頼度と区間の幅の関係

**サブセクション2** (10問) - 仮説検定の基礎
- 帰無仮説と対立仮説の理解
- p値の意味と解釈
- 有意水準と検定結果の解釈
- 検定結果の正しい表現

AIプロンプト例：
```
セクション: 推測統計の基礎
級: 3級
問題数: 10
詳細: 信頼区間の計算や仮説検定の結果を解釈する問題を作成してください。t検定・F検定・カイ二乗検定の具体的な手順は含めないでください。第一種・第二種の過誤も含めないでください。
```

---

### Section6: クロス集計表・実験計画 ★★（補助・10%）

**サブセクション1** (10問) - クロス集計表
- クロス集計表の読解
- 条件付き割合の計算
- 2つの変数の関係
- データの種類の識別（量的・質的）

**サブセクション2** (10問) - 標本調査と実験計画
- 母集団と標本の定義
- 無作為抽出の重要性
- 実験計画の基礎（無作為化、対応のあるデータ）
- バイアスの理解

AIプロンプト例：
```
セクション: クロス集計表・実験計画
級: 3級
問題数: 10
詳細: クロス集計表から割合を計算する問題や、標本調査の基本的な概念を問う問題を作成してください。
```

## 合計
- 6セクション
- 14サブセクション
- 140問（各サブセクション10問）

## 図表使用率
- Section1-3: 90%以上（記述統計、相関、データ変換）
- Section4: 50%程度（確率は計算問題が中心）
- Section5: 70%程度（ヒストグラムや散布図を使った推測統計）
- Section6: 80%程度（クロス集計表を使った問題）

**全体の図表使用率: 約85%** ← 実際の試験と一致
