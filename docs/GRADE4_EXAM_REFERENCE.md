## 統計検定4級の出題傾向まとめ

### 主要な出題範囲（頻出順）

1. **グラフの読解**（最重要）
   - ヒストグラム、箱ひげ図、円グラフ、棒グラフ、帯グラフ
   - 折れ線グラフ、幹葉図、ドットプロット
   - 累積相対度数グラフ、度数多角形
   - 複合グラフ、人口ピラミッド

2. **記述統計量**
   - 平均値、中央値、最頻値、範囲、四分位範囲
   - 度数分布表からの計算
   - 分布の歪みと代表値の関係
   - 外れ値の影響

3. **データの種類**
   - 量的データと質的データの識別

4. **割合・比率・指数**
   - 相対度数、百分率の計算
   - 実数と割合の区別
   - 指数の計算と解釈
   - 変化率の計算

5. **クロス集計表**
   - 2変数の関係
   - 条件付き割合

6. **確率**
   - 基本的な確率計算
   - 独立事象、条件付き確率
   - 復元抽出・非復元抽出

7. **標本調査・実験計画**
   - 抽出方法（無作為抽出、層化抽出など）
   - サンプリングバイアス
   - 無作為化

8. **公的統計**
   - 国勢調査の基礎知識

### 特徴
- 視覚的な資料（グラフ、表）の読解が中心
- 実生活に関連したデータが多い
- 計算問題は基礎的なレベル
- データの解釈と批判的思考を問う問題が含まれる
